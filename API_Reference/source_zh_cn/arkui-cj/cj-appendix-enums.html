API 参考

基础类型定义

自定义类型说明

AppResource

资源引用类型，用于设置组件属性的值。

可以通过@r或者@rawfile创建AppResource类型对象，不可以修改AppResource中的各属性的值。

  • @r(belonging.type.name)

    belonging: 系统资源或者应用资源，相应的取值为'sys'和'app'。 type：资源类型，支持'boolean'、'color'、'float'、'intarray'、'integer'、'pattern'、'plural'、'strarray'、'string'、'media'。 name：资源名称，在资源定义时确定。

  • @rawfile("filename")

    filename：工程中resources/rawfile目录下的文件名称

说明： 在引用资源类型时，注意其数据类型要与属性方法本身的类型一致，例如某个属性方法支持设置String | AppResource，那么在使用AppResource引用类型时，其数据类型也应当为String。

CJResource

继承自AppResource，用于兼容旧版本资源类型。

PluralResource

继承自AppResource，用于兼容旧版本资源类型。

AnimateParam

public struct AnimateParam {
    public init(
        public var duration!: Option<Int32> = 1000,
        public var tempo!: Option<Float32> = 1.0,
        public var curve!: Option<Curve> = Curve.EaseInOut,
        public var delay!: Option<Int32> = 0,
        public var iterations!: Option<Int32> = 1,
        public var playMode!: Option<PlayMode> = PlayMode.Normal,
        public var onFinish!: Option<() -> Unit> = Option.None,
        public var finishCallbackType!: Option<FinishCallbackType> = FinishCallbackType.REMOVED,
        public var expectedFrameRateRange!: Option<ExpectedFrameRateRange> = Option.None
    )
}

构造函数

public init(
    public var duration!: Option<Int32> = 1000,
    public var tempo!: Option<Float32> = 1.0,
    public var curve!: Option<Curve> = Curve.EaseInOut,
    public var delay!: Option<Int32> = 0,
    public var iterations!: Option<Int32> = 1,
    public var playMode!: Option<PlayMode> = PlayMode.Normal,
    public var onFinish!: Option<() -> Unit> = Option.None,
    public var finishCallbackType!: Option<FinishCallbackType> = FinishCallbackType.REMOVED,
    public var expectedFrameRateRange!: Option<ExpectedFrameRateRange> = Option.None
)
参数名参数类型必填默认值描述
durationOption<Int32>否默认值：1000动画持续时间，单位为毫秒。
设置小于0的值时按0处理。
tempoOption<Float32>否1.0动画播放速度，值越大动画播放越快，值越小播放越慢，为0时无动画效果。
当设置小于0的值时按值为1处理。
