尺寸设置

设置组件的宽高、边距。

函数

width(Length)

public func width(value: Length): This

设置组件自身的宽度，缺省时使用元素自身内容需要的宽度。若子组件的宽大于父组件的宽，则会画出父组件的范围。

参数名参数类型必填默认值描述
valueLength是-组件宽度。
单位：vp

width(CJResource)

public func width(value: CJResource): This
参数名参数类型必填默认值描述
valueCJResource是-组件宽度，引入系统资源或者应用资源中的尺寸。

width(Float64)

public func width(value: Float64): This
参数名参数类型必填默认值描述
valueFloat64是-组件宽度。
单位：vp

width(Int64)

public func width(value: Int64): This
参数名参数类型必填默认值描述
valueInt64是-组件宽度。
单位：vp

width<T>(Option<T>)

public func width<T>(value: Option<T>): This

设置组件自身的宽度，参数值为None时使用元素自身内容需要的宽度。

参数名参数类型必填默认值描述
valueOption<T>是-组件宽度。
T为Int64、Float64、Length、CJResource类型。

package ohos_app_cangjie_entry

@Entry
@Component
class EntryView {
    func build() {
        Column() {
            Text("margin and padding:").fontSize(12).fontColor(0xCCCCCC).width(90.percent)
            Row() {
            // 宽度80 ,高度80 ,外边距20（蓝色区域），内边距10（白色区域）
                Row() {
                    Row().size(width: 100.percent, height: 100.percent).backgroundColor(Color.GREEN)
                }
                .width(80)
                .height(80)
                .padding(10)
                .margin(20)
                .backgroundColor(Color.WHITE)
            }.backgroundColor(Color.BLUE)

            Text("constraintSize").fontSize(12).fontColor(0xCCCCCC).width(90.percent)
            Text("this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text.this is a Text")
            .width(90.percent)
            .constraintSize(maxWidth: 200.vp)

            Text("layoutWeight").fontSize(12).fontColor(0xCCCCCC).width(90.percent)
            // 父容器尺寸确定时，设置了layoutWeight的子元素在主轴布局尺寸按照权重进行分配，忽略本身尺寸设置。
            Row() {
                // 权重1，占主轴剩余空间1/3
                Text("layoutWeight(1)")
                .size(width: 30.percent, height: 110.vp).backgroundColor(0xFFEFD5).textAlign(TextAlign.Center)
                .layoutWeight(1)
                // 权重2，占主轴剩余空间2/3
                Text("layoutWeight(2)")
                .size(width: 30.percent, height: 110.vp).backgroundColor(0xF5DEB3).textAlign(TextAlign.Center)
                .layoutWeight(2)
                // 未设置layoutWeight属性，组件按照自身尺寸渲染
                Text("no layoutWeight")
                .size(width: 30.percent, height: 110.vp).backgroundColor(0xD2B48C).textAlign(TextAlign.Center)
            }.size(width: 90.percent, height: 140.vp).backgroundColor(0xAFEEEE)
            Text("calc:").fontSize(12).fontColor(0xCCCCCC).width(90.percent)
                Text("calc test")
                .fontSize(50)
                .fontWeight(FontWeight.Bold)
                .backgroundColor(0xFFFAF0)
                .textAlign(TextAlign.Center)
                .margin(50)
                .size(width: 90.percent, height: 10.percent)
        }.width(100.percent).margin(top: 5)
    }
