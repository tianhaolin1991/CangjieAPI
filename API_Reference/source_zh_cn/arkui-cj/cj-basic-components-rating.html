API 参考

Rating

一个提供在给定范围内选择评分的组件。

说明：

Rating暂不支持混合开发V1场景, 推荐使用混合开发V2。

子组件

无

构造函数

init(Float64, Bool)

public init(rating!: Float64, indicator!: Bool = false)

构造一个在给定范围内选择评分的组件。

参数名参数类型必填默认值描述
ratingFloat64是0设置并接收评分值。取值范围： [0, stars] ，小于0取0，大于stars取最大值stars。
indicatorBool否false设置评分组件作为指示器使用，不可改变评分。默认值为false时，可进行评分。

init(Int64, Bool)

public init(rating!: Int64, indicator!: Bool = false)

构造一个在给定范围内选择评分的组件。

参数名参数类型必填默认值描述
ratingInt64是0设置并接收评分值。取值范围： [0, stars] ，小于0取0，大于stars取最大值stars。
indicatorBool否false设置评分组件作为指示器使用，不可改变评分。默认值为false时，可进行评分。

函数

stars(Int32)

public func stars(value: Int32): This

设置评分总数。设置为小于等于0的值时，按默认值显示。

参数名参数类型必填默认值描述
valueInt32是5设置评分总数。

stepSize(Float64)

public func stepSize(value: Float64): This

设置操作评级的步长。设置为小于0.1的值时，按默认值显示。

参数名参数类型必填默认值描述
valueFloat64是0.5操作评级的步长。取值范围：[0.1, stars] 。

stepSize(Int64)

public func stepSize(value: Int64): This

设置操作评级的步长。设置为小于0.1的值时，按默认值显示。

参数名参数类型必填默认值描述
valueInt64是0.5操作评级的步长。取值范围：[0.1, stars] 。

starStyle(String, String, String)

public func starStyle(backgroundUri!: String, foregroundUri!: String, secondaryUri!: String = backgroundUri): This

设置评分的样式。该属性所支持的图片类型能力参考Image组件。支持加载本地图片和网络图片，暂不支持PixelMap类型和Resource资源。 默认图片加载方式为异步，暂不支持同步加载。

说明：

  • rating宽高为[width, height]时，单个图片的绘制区域为[width / stars, height]。
  • 为了指定绘制区域为方形，建议自定义宽高时采取[height * stars, height], width = height * stars的方式。
  • backgroundUri或者foregroundUri设置为undefined或者空字符串时，rating会选择加载系统默认星型图源。
  • secondaryUri不设置或者设置的值为undefined或者空字符串时，优先设置为backgroundUri，效果上等同于只设置了foregroundUri、backgroundUri。
参数名参数类型必填默认值描述
backgroundUriString是-未选中的星级的图片链接，可由用户自定义或使用系统默认图片。
foregroundUriString是-选中的星级的图片路径，可由用户自定义或使用系统默认图片。
secondaryUriString否-部分选中的星级的图片路径，可由用户自定义或使用系统默认图片
