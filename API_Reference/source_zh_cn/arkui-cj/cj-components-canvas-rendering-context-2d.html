CanvasRenderingContext2D

使用RenderingContext在Canvas组件上进行绘制，绘制对象可以是矩形、文本、图片等。

说明：

CanvasRenderingContext2D暂不支持混合开发V1场景, 推荐使用混合开发V2。

构造函数

init(RenderingContextSettings)

public init(settings: RenderingContextSettings)

使用settings构造一个CanvasRenderingContext2D对象。

参数名参数类型必填默认值描述
settingsRenderingContextSettings是-用来配置CanvasRenderingContext2D对象的参数，见RenderingContextSettings。

函数

fillStyle(Color)

public func fillStyle(color: Color): Unit

指定绘制的填充色。

参数名参数类型必填默认值描述
colorColor是-表示设置填充区域的颜色。

fillStyle(UInt32)

public func fillStyle(color: UInt32): Unit

指定绘制的填充色。

参数名参数类型必填默认值描述
colorUInt32是-表示设置填充区域的颜色。

fillStyle(CanvasGradient)

public func fillStyle(color: CanvasGradient): Unit

指定绘制的填充色。

参数名参数类型必填默认值描述
colorCanvasGradient是-表示渐变对象，使用createLinearGradient方法创建。
  • 示例：
package ohos_app_cangjie_entry
import ohos.base.*
import ohos.component.*
import ohos.state_manage.*
import ohos.state_macro_manage.*

@Entry
@Component
class EntryView {
    var settings: RenderingContextSettings = RenderingContextSettings(antialias: true)
    var context: CanvasRenderingContext2D = CanvasRenderingContext2D(this.settings)

    func build() {
        Flex(
            FlexParams(direction: FlexDirection.Column, alignItems: ItemAlign.Center,
            justifyContent: FlexAlign.Center)) {
            Canvas(this.context).width(100.percent).height(100.percent).backgroundColor(0xffff00).onReady(
                {
                    =>
                    this.context.fillStyle(0x0000ff)
                    this.context.fillRect(20, 20, 150, 100)
                }
            )
        }.width(100.percent).height(100.percent)
    }
}

lineCap(LineCapStyle)

public func lineCap(value: LineCapStyle): Unit

指定线端点的样式。

参数名参数类型必填默认值描述
valueLineCapStyle是Butt- 'Butt'：线端点以方形结束。
- 'Round'：线端点以圆形结束。
- 'Square'：线端点以方形结束，该样式下会增加一个长度和线段厚度相同，宽度是线段厚度一半的矩形。

lineJoin(LineJoinStyle)

public func lineJoin(value: LineJoinStyle): Unit

指定线段间相交的交点样式。

参数名参数类型必填默认值描述
valueLineJoinStyle是Miter- 'Round'：在线段相连处绘制一个扇形，扇形的圆角半径是线段的宽度。
- 'Bevel'：在线段相连处使用三角形为底填充， 每个部分矩形拐角独立。
- 'Miter'：在相连部分的外边缘处进行延伸，使其相交于一点，形成一个菱形区域，该属性可以通过设置miterLimit属性展现效果。

miterLimit(Float64)

public func miterLimit(limit: Float64): Unit
