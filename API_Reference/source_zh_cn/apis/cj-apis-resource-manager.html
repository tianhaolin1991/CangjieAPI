ohos.resource_manager（资源管理）

资源管理模块，根据当前configuration：语言、区域、横竖屏、Mcc（移动国家码）和Mnc（移动网络码）、Device capability（设备类型）、Density（分辨率）提供获取应用资源对象读取接口。

导入模块

import ohos.resource_manager.*

class Configuration

public class Configuration <: ToString {
    public Configuration(
        public let direction: Direction,
        public let locale: String
    ) {}
}

表示当前设备的配置。

系统能力：SystemCapability.Global.ResourceManager

参数：

名称类型可读可读说明
directionDirection是否当前设备屏幕方向。
localeString是否当前系统语言。

func toString()

public func toString(): String

获取当前 Configuration 的信息，以字符串表示。

返回值：

类型说明
String当前 Configuration 的信息。

class ResourceManager

public class ResourceManager <: RemoteData & ResourceManagerInterface {
    public static func getResourceManager(context: StageContext): ResourceManager
}

提供访问应用资源的能力。

说明：

  • 资源文件在工程的resources目录中定义，资源ID可通过@r(资源地址).id的方式获取，例如@r(app.string.test).id。

  • 对于本应用包资源，通过指定资源ID或资源名进行访问。对于应用内跨包资源，通过指定Resource对象进行访问。

static func getResourceManager(StageContext)

public static func getResourceManager(context: StageContext): ResourceManager

根据上下文，获取资源管理对象。

参数：

参数名类型必填说明
contextStageContext是上下文。

返回值：

类型说明
ResourceManager资源管理对象。

示例：

// 步骤 1，在工程中的 main_ability.cj 中加入对 context 的赋值。
class MainAbility <: Ability {
    static var abilityContext: Option<AbilityContext> = None
    public override func onWindowStageCreate(windowStage: WindowStage): Unit {
      ...
      abilityContext = this.context
      ...
    }
}

// 步骤2，通过 AbilityContext 获取 StageContext
let stageContext = getStageContext(MainAbility.abilityContext.getOrThrow())

// 步骤3，获取 ResourceManager 对象
let resMgr = ResourceManager.getResourceManager(stageContext)

func getString(Int32)

public func getString(resId: Int32): String

获取资源ID对应的字符串资源。

系统能力：SystemCapability.Global.ResourceManager

参数：

参数名类型必填说明
resIdInt32是资源ID。

返回值：

类型说明
String资源ID对应的字符串资源。

错误码：

以下错误码的详细介绍请参见资源管理错误码。

错误码ID错误信息
9001001If the resId invalid.
9001002If the resource not found by resId.
9001006If the resource re-ref too much.

示例：

import ohos.resource_manager.*
import cj_res_entry.app

let resourceManager = ResourceManager.getResourceManager(stageContext)
let resource = @r(app.string.test)

let value = try {
    resourceManager.getString(Int32(resource.id))
} catch (e: BusinessException) {
    let code = e.code
    let message = e.message
    AppLog.info("getString failed, error code: ${code}, message: ${message}.")
    ""
}
