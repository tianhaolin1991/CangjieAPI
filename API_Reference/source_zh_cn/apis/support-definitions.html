系统公共事件定义

本文档提供系统所定义的公共事件类型的索引。 公共事件类型均以静态常量定义在ohos.commonEventManager模块的Support中，类型均为 String。

系统能力： SystemCapability.Notification.CommonEvent

  • COMMON_EVENT_BOOT_COMPLETED 表示用户已完成引导并加载系统的公共事件。

  • COMMON_EVENT_LOCKED_BOOT_COMPLETED （预留事件，暂未支持）表示用户已完成引导，系统已加载，但屏幕仍锁定的公共事件。

    • 常量值：usual.event.LOCKED_BOOT_COMPLETED
    • 订阅者所需权限：ohos.permission.RECEIVER_STARTUP_COMPLETED

  • COMMON_EVENT_SHUTDOWN 表示设备正在关闭的公共事件。

  • COMMON_EVENT_BATTERY_CHANGED 表示电池充电状态、电平和其他信息发生变化的公共事件。

  • COMMON_EVENT_BATTERY_LOW 表示电池电量低的公共事件。

  • COMMON_EVENT_BATTERY_OKAY 表示电池退出低电平状态的公共事件。

  • COMMON_EVENT_POWER_CONNECTED 表示设备连接到外部电源的公共事件。

  • COMMON_EVENT_POWER_DISCONNECTED 表示设备与外部电源断开的公共事件。

  • COMMON_EVENT_SCREEN_OFF 表示设备屏幕关闭且设备处于睡眠状态的公共事件。

  • COMMON_EVENT_SCREEN_ON 表示设备屏幕打开且设备处于交互状态的公共事件。

  • COMMON_EVENT_THERMAL_LEVEL_CHANGED 表示设备热状态的公共事件。

  • COMMON_EVENT_TIME_TICK 表示系统时间更改的公共事件。

  • COMMON_EVENT_TIME_CHANGED 表示设置系统时间的公共事件。

  • COMMON_EVENT_DATE_CHANGED （预留事件，暂未支持）表示系统日期已更改的公共事件。

    • 常量值：usual.event.DATE_CHANGED
    • 订阅者所需权限：无

  • COMMON_EVENT_TIMEZONE_CHANGED 表示系统时区更改的公共事件。

  • COMMON_EVENT_CLOSE_SYSTEM_DIALOGS （预留事件，暂未支持）表示用户关闭临时系统对话框的公共事件。

    • 常量值：usual.event.CLOSE_SYSTEM_DIALOGS
    • 订阅者所需权限：无

  • COMMON_EVENT_PACKAGE_ADDED 表示设备上已安装新应用包的公共事件。

  • COMMON_EVENT_PACKAGE_REPLACED （预留事件，暂未支持）表示已安装的应用程序包的新版本已替换设备上的旧版本的公共事件。

    • 常量值：usual.event.PACKAGE_REPLACED
    • 订阅者所需权限：无

  • COMMON_EVENT_MY_PACKAGE_REPLACED （预留事件，暂未支持）表示应用程序包的新版本已取代前一个版本的公共事件。

    • 常量值：usual.event.MY_PACKAGE_REPLACED
    • 订阅者所需权限：无

  • COMMON_EVENT_PACKAGE_REMOVED 表示已从设备卸载已安装的应用程序，但应用程序数据保留的公共事件。

  • COMMON_EVENT_BUNDLE_REMOVED （预留事件，暂未支持）表示已从设备中卸载已安装的捆绑包，但应用程序数据仍保留的公共事件。

    • 常量值：usual.event.BUNDLE_REMOVED
    • 订阅者所需权限：无

  • COMMON_EVENT_PACKAGE_FULLY_REMOVED （预留事件，暂未支持）表示已从设备中完全卸载已安装的应用程序（包括应用程序数据和代码）的公共事件。

    • 常量值：usual.event.PACKAGE_FULLY_REMOVED
    • 订阅者所需权限：无

  • COMMON_EVENT_PACKAGE_CHANGED 表示应用包已更改的公共事件（例如，包中的组件已启用或禁用）。

  • COMMON_EVENT_PACKAGE_RESTARTED 表示用户重启应用包并杀死其所有进程的公共事件。

  • COMMON_EVENT_PACKAGE_DATA_CLEARED 表示用户清除应用包数据的公共事件。

  • COMMON_EVENT_PACKAGE_CACHE_CLEARED 表示用户清除应用包缓存数据的公共事件。

  • COMMON_EVENT_PACKAGES_SUSPENDED （预留事件，暂未支持）表示应用包已挂起的公共事件。

    • 常量值：usual.event.PACKAGES_SUSPENDED
    • 订阅者所需权限：无

  • COMMON_EVENT_PACKAGES_UNSUSPENDED （预留事件，暂未支持）表示应用包未挂起的公共事件。

    • 常量值：usual.event.PACKAGES_UNSUSPENDED
    • 订阅者所需权限：无

  • COMMON_EVENT_MY_PACKAGE_SUSPENDED 表示应用包被挂起的公共事件。

    • 常量值：usual.event.MY_PACKAGE_SUSPENDED
    • 订阅者所需权限：无

  • COMMON_EVENT_MY_PACKAGE_UNSUSPENDED 表示应用包未挂起的公共事件。
