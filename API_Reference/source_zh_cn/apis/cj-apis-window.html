API 参考

ohos.window（窗口）

窗口提供管理窗口的一些基础能力，包括对当前窗口的创建、销毁、各属性设置，以及对各窗口间的管理调度。

该模块提供以下窗口相关的常用功能：

  • Window：当前窗口实例，窗口管理器管理的基本单元。

说明

ohos.window暂不支持在混合开发场景中使用。

导入模块

import ohos.window.*

enum WindowType

public enum WindowType {
    | TYPE_APP
    | TYPE_SYSTEM_ALERT
    | TYPE_FLOAT
    | TYPE_DIALOG
}

窗口类型枚举。

系统能力： SystemCapability.WindowManager.WindowManager.Core

名称说明
TYPE_APP表示应用子窗口。
TYPE_SYSTEM_ALERT表示系统告警窗口。
TYPE_FLOAT表示悬浮窗。
TYPE_DIALOG表示模态窗口。

enum AvoidAreaType

public enum AvoidAreaType {
    | TYPE_SYSTEM
    | TYPE_CUTOUT
    | TYPE_SYSTEM_GESTURE
    | TYPE_KEYBOARD
    | TYPE_NAVIGATION_INDICATOR
}

窗口内容需要规避区域的类型枚举。

系统能力： SystemCapability.WindowManager.WindowManager.Core

名称说明
TYPE_SYSTEM表示系统默认区域。一般包括状态栏、导航栏，各设备系统定义可能不同。
TYPE_CUTOUT表示刘海屏区域。
TYPE_SYSTEM_GESTURE表示手势区域。
TYPE_KEYBOARD表示软键盘区域。
TYPE_NAVIGATION_INDICATOR表示导航条区域。

enum Orientation

public enum Orientation {
    | UNSPECIFIED
    | PORTRAIT
    | LANDSCAPE
    | PORTRAIT_INVERTED
    | LANDSCAPE_INVERTED
    | AUTO_ROTATION
    | AUTO_ROTATION_PORTRAIT
    | AUTO_ROTATION_LANDSCAPE
    | AUTO_ROTATION_RESTRICTED
    | AUTO_ROTATION_PORTRAIT_RESTRICTED
    | AUTO_ROTATION_LANDSCAPE_RESTRICTED
    | LOCKED
}

窗口显示方向类型枚举。

系统能力： SystemCapability.WindowManager.WindowManager.Core

名称说明
UNSPECIFIED表示未定义方向模式，由系统判定。
PORTRAIT表示竖屏显示模式。
LANDSCAPE表示横屏显示模式。
PORTRAIT_INVERTED表示反向竖屏显示模式。
LANDSCAPE_INVERTED表示反向横屏显示模式。
AUTO_ROTATION表示传感器自动旋转模式。
AUTO_ROTATION_PORTRAIT表示传感器自动竖向旋转模式。
AUTO_ROTATION_LANDSCAPE表示传感器自动横向旋转模式。
AUTO_ROTATION_RESTRICTED表示受开关控制的自动旋转模式。
AUTO_ROTATION_PORTRAIT_RESTRICTED表示受开关控制的自动竖向旋转模式。
AUTO_ROTATION_LANDSCAPE_RESTRICTED表述受开关控制的自动横向旋转模式。
LOCKED表示锁定模式。

enum ColorSpace

public enum ColorSpace {
    | DEFAULT
    | WIDE_GAMUT
}

色域模式。

系统能力： SystemCapability.WindowManager.WindowManager.Core

名称说明
DEFAULT默认SRGB色域模式。
WIDE_GAMUT广色域模式。

enum WindowEventType

public enum WindowEventType {
    | WINDOW_SHOWN
    | WINDOW_ACTIVE
    | WINDOW_INACTIVE
    | WINDOW_HIDDEN
    | WINDOW_DESTROYED
}

窗口生命周期。

系统能力： SystemCapability.WindowManager.WindowManager.Core

名称说明
WINDOW_SHOWN切到前台。
WINDOW_ACTIVE切到前台。
WINDOW_INACTIVE失焦状态。
WINDOW_HIDDEN失焦状态。
WINDOW_DESTROYED窗口销毁。

enum WindowStageEventType

public enum WindowStageEventType {
    | SHOWN
    | ACTIVE
    | INACTIVE
    | HIDDEN
    | RESUMED
    | PAUSED
}

WindowStage生命周期。

模型约束： 此接口仅可在Stage模型下使用。

系统能力： SystemCapability.WindowManager.WindowManager.Core

名称说明
SHOWN切到前台。
ACTIVE获焦状态。
INACTIVE失焦状态。
HIDDEN切到后台。
RESUMED前台可交互状态。前台应用进入多任务为不可交互状态，继续返回前台时恢复可交互状态。
PAUSED前台不可交互状态。前台应用进入多任务为不可交互状态，继续返回前台时恢复可交互状态。

enum WindowStatusType

public enum WindowStatusType {
    | UNDEFINED
    | FULL_SCREEN
    | MAXIMIZE
    | MINIMIZE
    | FLOATING
    | SPLIT_SCREEN
}

窗口模式枚举。

系统能力： SystemCapability.Window.SessionManager

名称说明
UNDEFINED表示APP未定义窗口模式。
FULL_SCREEN表示APP全屏模式。
MAXIMIZE表示APP窗口最大化模式。
MINIMIZE表示APP窗口最小化模式。
FLOATING表示APP自由悬浮形式窗口模式。
SPLIT_SCREEN表示APP分屏模式。

struct SystemBarProperties

public struct SystemBarProperties {
    public SystemBarProperties(
        public var statusBarColor!: String = "#66000000",
